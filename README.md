# 🚀 SkillSwap – MicroSkill Exchange Platform

SkillSwap is a full-stack MicroSkill Exchange Platform where users can exchange skills, post problems, offer solutions, and earn coins through collaborative learning.

Built with a scalable architecture using the MERN Stack and modern development practices.

## 📌 Project Overview

SkillSwap enables learners and instructors to exchange micro-skills in a structured way. Users can:

- Post skill-based problems
- Bid on tasks
- Accept instructors
- Complete tasks and earn coin rewards
- Track performance via leaderboard

---

## ✨ Key Features

### 🔐 Authentication & Authorization
- JWT-based authentication
- Role-based access (Learner / Instructor / Admin)
- Secure password hashing
- Protected routes

### 📚 Problem Posting System
- Create, edit, and delete problem posts
- Set reward coins
- Skill requirement filtering
- Status management (OPEN / IN_PROGRESS / COMPLETED)

### 💰 Coin Reward System
- Earn coins after task completion
- Automatic coin updates
- Performance tracking

### 🏆 Leaderboard System
- Instructor ranking based on coins
- Top instructors fetching with limit
- Dynamic leaderboard API

### 📊 Dashboard
- User profile management
- Task history
- Performance overview
